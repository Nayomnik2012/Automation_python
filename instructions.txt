git init (инициализируем директорию для работы с гитом)
git add .   добавляем все файлы(можно указать имя)
git commit -m ""  делаем комит
git checkout xxx  - меняем ветку
git checkout -b <....> - создаем ветку
git branch - список всех веток()
git diff - показывает изменения
git merge project_branch - находясь на ветке выливаем изменения в указанную
